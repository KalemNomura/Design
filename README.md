#Proyecto R3#

Este proyecto está hecho con React, TypeScript y TailwindCSS.

Usa React Hooks como useState, manejo de errores en imágenes con onError y tipos de TypeScript para las props de imágenes.

Para empezar necesitas tener Node.js instalado. Puedes descargarlo desde https://nodejs.org/. 
Después de instalarlo, comprueba que funciona con node -v y npm -v.

Luego descarga el proyecto con git clone (url del repo) y entra en la carpeta con cd nombre-del-proyecto.

Después instala las dependencias con npm install.

Para ejecutar el proyecto usa npm run dev y abre el enlace que aparece en la terminal
